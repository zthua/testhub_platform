# 数据工厂使用说明

## 📖 概述

数据工厂是 TestHub 平台的核心功能模块之一，提供60个实用工具，涵盖字符处理、编码转换、随机数据生成、加密解密、测试数据生成、JSON处理和Crontab表达式管理等多个场景。所有工具生成的数据都可以添加标签，方便在接口测试和UI测试中引用。

## 🎯 核心特性

### 1. 标签系统
- **多标签支持**：每条数据记录可以添加多个标签
- **标签分类**：支持按标签筛选和管理数据
- **数据引用**：在接口测试和UI测试中可以引用带标签的数据
- **灵活查询**：支持按标签名称、工具分类、工具名称等多维度查询

## 🚀 快速开始

### 1. 访问数据工厂

登录系统后，在左侧导航栏找到"数据工厂"菜单，点击进入。

### 2. 选择工具

在工具分类中选择需要的工具类别，然后选择具体的工具。

### 3. 输入参数

根据工具要求输入相应的参数，例如：
- **字符串处理**：输入要处理的文本
- **编码转换**：输入要转换的内容
- **随机生成**：设置生成数量和参数
- **JSON处理**：输入JSON字符串

### 4. 执行工具

点击"执行"按钮，工具会立即处理并返回结果。

### 5. 查看结果

- 执行结果会显示在结果区域
- 某些工具（如JSON格式化）会提供树形展示和展开/折叠功能
- 图片类工具（条形码、二维码）会显示图片预览
- 长数据支持左右滚动查看

### 6. 添加标签（可选）

在执行结果下方，可以输入标签（多个标签用逗号分隔），然后点击"保存"按钮。保存后的数据可以在接口测试和UI测试中引用。

## 📋 详细功能说明

### 字符工具

#### 去除空格和换行
- **功能**：去除文本中的所有空格、制表符、换行符等空白字符

- **功能**：替换文本中的指定字符串
- **输入**：

  - 替换后的字符串

- **输出**：替换后的文本和替换次数
- **使用场景**：批量替换文本内容

#### 字符串转义
- **功能**：将字符串转义为特定格式
- **输入**：
  - 原文本
  - 转义类型（JSON/HTML/URL/XML）
- **输出**：转义后的字符串
- **使用场景**：防止XSS攻击、URL编码等

#### 字符串反转义
- **功能**：将转义后的字符串还原
- **输入**：
  - 转义后的文本
  - 反转义类型（JSON/HTML/URL/XML）
- **输出**：还原后的字符串
- **使用场景**：解析转义后的数据

#### 字数统计
- **功能**：统计文本的详细信息
- **输入**：要统计的文本
- **输出**：
  - 总长度
  - 中文字符数
  - 英文单词数
  - 数字个数
  - 标点符号数

- **使用场景**：文本分析、内容统计

#### 文本对比
- **功能**：对比两段文本的差异
- **输入**：

- **输出**：

- **使用场景**：代码对比、文档版本对比

#### 正则表达式测试

  - 匹配数量

#### 大小写转换
- **功能**：转换文本的大小写
- **输入**：

- **使用场景**：文本格式化

#### 字符串格式化
- **功能**：对字符串进行格式化处理

## 🔧 常见问题

### Q1: 生成的图片在哪里？
A: 生成的条形码和二维码图片保存在服务器的`static_files/img`目录下，可以通过URL直接访问。

### Q2: 如何在接口测试中使用数据工厂的数据？
A: 在请求参数、断言、前置条件中，点击"引用数据工厂"按钮（魔棒图标），选择需要的数据即可。

### Q3: 标签有什么作用？
A: 标签用于分类和管理数据，方便在接口测试和UI测试中快速引用特定类型的数据。

### Q4: 数据工厂的数据可以修改吗？
A: 数据工厂保存的是工具执行的结果，不支持修改。如需修改，请重新执行工具并保存新的结果。

### Q5: JSON格式化支持多大的文件？
A: JSON格式化工具支持较大的JSON文件，但建议文件大小不超过10MB，以保证性能。

### Q6: 生成的测试数据是真实的吗？
A: 测试数据工具生成的数据是模拟的真实数据，格式符合规范，但不是真实用户数据。

### Q7: 加密工具的数据安全吗？
A: 所有加密操作在服务器端执行，数据不会保存到数据库（除非用户主动保存）。建议不要在数据工厂中处理真实的敏感信息。

### Q8: 如何批量删除历史记录？
A: 目前不支持批量删除，需要逐条删除。如需清空所有记录，请联系管理员。

### Q9: 数据预览支持滚动吗？
A: 是的，数据预览窗口支持左右滚动，可以查看长数据的完整内容。

### Q10: UI测试的变量助手包含所有函数吗？
A: 是的，UI测试的变量助手已经更新，包含数据工厂的所有60个工具函数。

## 📝 API接口

数据工厂提供以下API接口：

### 执行工具
```
POST /api/data-factory/
```

**请求体**：
```json
{
  "tool_name": "format_json",
  "tool_category": "json",
  "tool_scenario": "data_validation",
  "input_data": {
    "json_str": "{\"name\":\"test\"}",
    "indent": 2,
    "sort_keys": false
  },
  "is_saved": true,
  "tags": ["test_data", "json"]
}
```

**响应**：
```json
{
  "results": {...},
  "count": 1,
  "total_requested": 1
}
```

### 获取历史记录
```
GET /api/data-factory/
```

**查询参数**：
- `tool_name`：工具名称
- `tool_category`：工具分类
- `tags__contains`：标签（模糊匹配）
- `page`：页码
- `page_size`：每页数量

GET /api/data-factory/tags/
